Docgen - AI 기반 프로젝트 문서 자동 생성 서비스

Docgen은 개발 프로젝트의 문서화 작업을 자동화하는 AI 기반 서비스입니다. 프로젝트 코드와 요구사항을 분석하여 API 문서, 사용자 가이드, 기술 문서 등을 자동으로 생성합니다.

LangChain과 CrewAI를 활용한 멀티 에이전트 시스템을 구현하여, 복잡한 문서 생성 프로세스를 여러 AI 에이전트가 협업하여 처리하도록 설계했습니다. TypeScript를 사용하여 타입 안정성을 확보하고, MongoDB를 통해 비정형 데이터를 효율적으로 관리했습니다.

주요 기능

  • 코드 기반 API 문서 자동 생성
  • 프로젝트 요구사항 분석 및 문서화
  • 멀티 에이전트 협업 시스템 (CrewAI)
  • 문서 템플릿 관리 및 커스터마이징
  • 문서 버전 관리 및 히스토리 추적
  • 실시간 문서 생성 진행 상황 모니터링

기술 스택

  • Backend: Node.js, Python, FastAPI, PostgreSQL, MongoDB
  • Frontend: React, TypeScript
  • AI: LLM(OpenAI), LangChain, CrewAI
  • 기타: RESTful API, WebSocket, 멀티 에이전트 시스템

AI 기술 적용

LangChain을 활용하여 코드 분석, 문서 생성, 품질 검증 등의 여러 단계를 체인으로 연결하고, CrewAI의 멀티 에이전트 시스템을 통해 각 단계를 전문 에이전트가 처리하도록 구현했습니다. 이를 통해 정확하고 일관성 있는 문서를 자동으로 생성할 수 있습니다.

프로젝트 특징

Python FastAPI와 Node.js를 함께 사용하여 각 언어의 장점을 활용했으며, PostgreSQL과 MongoDB를 혼합하여 정형/비정형 데이터를 효율적으로 관리했습니다. TypeScript를 도입하여 프론트엔드 개발의 안정성을 높였습니다.
