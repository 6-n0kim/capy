InPick — AI 기반 여행 일정 생성 및 추천 시스템

사용자의 취향과 위치 정보를 분석하여 개인화된 여행 코스를 설계해주는 올인원 여행 플랫폼입니다.

1. 시스템 아키텍처 (System Architecture)

FastAPI를 중심으로 다양한 AI 모델과 3종류의 데이터베이스(RDB, NoSQL, Vector DB)가 유기적으로 연결된 구조입니다.

2. 주요 기능 및 화면 (UI/UX)

맞춤 장소 추천

지역, 기간, 동행자, 테마를 단계별로 입력받아 CBF, CF 추천 알고리즘을 통해 최적의 장소를 제안합니다.

시퀀스 추천 + 지도 기반 경로 확인

시퀀스 추천을 통해 생성된 일정의 동선을 Kakao Map과 TMAP API로 시각화하여 보여줍니다.

3. AI 추천 프로세스 (Recommendation Flow)

사용자 입력부터 최종 일정 생성까지 5가지 추천 알고리즘이 복합적으로 작용합니다.

4. 데이터베이스 설계 (ERD)

사용자 정보와 장소 데이터는 관계형 DB에, 세션과 로그는 NoSQL에 분산 저장합니다.

6. 트러블 슈팅

API 호출 시 JSON 데이터가 정상적으로 전달되지 않는 문제를 해결하기 위해 Axios 인터셉터를 사용하여 요청/응답을 로깅하고, 데이터를 확인
JWT 인증 과정에서 토큰이 정상적으로 전달되지 않는 문제를 해결하기 위해 interceptor를 사용하여 토큰을 추가
CORS 문제를 해결하기 위해 프론트 작업을 하는 동기와 정보 공유 후 해결
