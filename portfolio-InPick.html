InPick - 인스타그램 데이터를 통한 여행 추천 서비스

InPick은 인스타그램의 여행 관련 게시물을 분석하여 개인화된 여행지 추천을 제공하는 AI 기반 추천 시스템입니다. RAG(Retrieval-Augmented Generation) 기술과 협업 필터링 알고리즘을 결합하여 정확하고 개인화된 여행지 추천을 구현했습니다.

KDT AI 교육 과정의 최종 프로젝트로, 머신러닝과 딥러닝 기법을 실제 서비스에 적용했습니다. 인스타그램 데이터를 크롤링하고 처리하며, 벡터 데이터베이스(FAISS)를 활용한 유사도 검색과 추천 알고리즘을 구현했습니다.

주요 기능

  • 인스타그램 여행 게시물 크롤링 및 전처리
  • RAG 기반 여행지 정보 검색 및 추천
  • 협업 필터링(CBF, ALS) 기반 맞춤형 추천
  • 딥러닝 모델을 통한 이미지 및 텍스트 분석
  • 사용자 선호도 기반 개인화 추천
  • 여행지 상세 정보 및 리뷰 제공

기술 스택

  • Backend: Python, FastAPI, PostgreSQL, MongoDB, FAISS
  • Frontend: React, TypeScript
  • AI/ML: RAG(OpenAI), ML - Implicit [CBF, ALS], DL - PyTorch
  • 기타: 데이터 크롤링, 벡터 검색, 추천 시스템

AI/ML 기술 적용

RAG (Retrieval-Augmented Generation): 여행지에 대한 최신 정보를 검색하고, 이를 기반으로 개인화된 추천 설명을 생성합니다.

협업 필터링: 콘텐츠 기반 필터링(CBF)과 행렬 분해 알고리즘(ALS)을 활용하여 사용자의 선호도와 유사한 여행지에 관심이 있는 다른 사용자들의 패턴을 분석합니다.

딥러닝: PyTorch를 사용하여 이미지와 텍스트를 분석하고, 여행지의 특징을 추출하여 추천 품질을 향상시킵니다.

프로젝트 특징

FAISS 벡터 데이터베이스를 활용하여 대규모 이미지 및 텍스트 데이터에서 유사도를 빠르게 검색할 수 있도록 구현했습니다. PostgreSQL과 MongoDB를 혼합하여 정형/비정형 데이터를 효율적으로 관리하며, FastAPI를 통해 고성능 API 서버를 구축했습니다.

데이터 처리

인스타그램에서 여행 관련 게시물을 수집하고, 이미지와 텍스트를 전처리하여 머신러닝 모델에 적합한 형태로 변환합니다. 벡터화된 데이터는 FAISS에 저장되어 실시간 유사도 검색이 가능합니다.
