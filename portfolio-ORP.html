(주) ORP 연구소 구인구직 웹 서비스

ORP 연구소의 구인구직 웹 서비스입니다. 직무별 검색, 지원서 관리, 채용 프로세스 추적 등 효율적인 인재 채용을 지원하는 시스템입니다.

Spring Framework와 JSP를 활용한 전통적인 MVC 패턴의 웹 애플리케이션으로, 회사에서 진행한 두 번째 프로젝트입니다. MariaDB를 사용하여 데이터 관리를 했으며, 사용자 경험을 개선하기 위한 다양한 기능을 구현하고 이전 코드를 리팩토링했습니다.

기술 스택

  • Backend: Java, Spring Framework, MariaDB
  • Frontend: JSP, JavaScript, HTML/CSS
  • 기타: MVC 패턴, RESTful API

담당 역할

  • 기존 코드 분석 및 오류 수정·기능 복구
  • 구인공고 상태 분기 및 관리 로직 구현
  • 구인공고 CRUD 기능 개발 (등록, 수정, 삭제, 조회)
  • RESTful API 구조 개선 및 응답 데이터 표준화

프로젝트 경험

ORP 연구소의 구인구직 프로젝트는 기존 코드 기반으로 진행되었으나 로직 일관성이 부족하고 오류가 다수 존재하는 상태였습니다. 이에 기존 코드를 분석하며 기능별 문제를 식별하고, 비정상적으로 동작하던 구인공고 관리 기능을 안정화했습니다. 특히 공고 상태 분기 처리 로직을 재설계하여 관리자·사용자 모두에게 일관된 데이터 흐름이 유지되도록 개선했습니다. 비록 프로젝트는 외부 사정으로 중도 종료되었지만, 복잡한 레거시 코드 분석과 오류 수정 경험을 통해 실무 환경에서의 문제 해결 능력과 유지보수 역량을 크게 향상시킬 수 있었습니다.
