OnSpot - 맵 기반 SNS 서비스

OnSpot은 지도를 기반으로 한 위치 기반 소셜 네트워크 서비스입니다. 사용자가 특정 장소에서의 경험과 사진을 공유하고, 다른 사용자들의 추천을 받을 수 있는 플랫폼입니다.

자바 개발자 양성 교육 과정에서 진행한 첫 번째 팀 프로젝트로, Spring Framework와 JSP를 활용한 웹 애플리케이션 개발 경험을 쌓았습니다. 카카오 맵 API를 활용하여 지도 서비스를 구현했으며, 사용자 인증 및 게시물 관리 기능을 개발했습니다.

주요 기능

  • 지도 기반 위치 표시 및 검색
  • 장소별 게시물 작성 및 공유
  • 사진 업로드 및 갤러리 기능
  • 댓글 및 좋아요 기능
  • 회원 관리 및 프로필 설정
  • 팔로우 및 팔로워 시스템

기술 스택

  • Backend: Java, Spring Framework, Oracle Database
  • Frontend: JSP, JavaScript, HTML/CSS
  • API: 카카오 맵 API
  • 기타: MVC 패턴, 세션 관리

프로젝트 경험

이 프로젝트를 통해 Spring Framework의 기본 구조와 동작 원리를 학습했으며, 웹 애플리케이션의 전체 개발 프로세스를 경험했습니다. 팀 프로젝트를 통해 협업 방법과 코드 리뷰 프로세스를 이해했습니다.
