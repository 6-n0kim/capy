On Spot — 위치 기반 소셜 미디어 플랫폼

내 주변의 경험을 공유하고, 지도 위에서 새로운 사람들의 이야기를 발견하는 공간

On Spot은 지도를 기반으로 한 위치 기반 소셜 네트워크 서비스입니다. 사용자가 특정 장소에서의 경험과 사진을 공유하고, 다른 사용자들의 추천을 받을 수 있는 플랫폼입니다.

문제 정의

기존 SNS에서는 게시물이 시간순으로만 정렬되어 "지금 이 장소 주변에서 무슨 일이 있는가"를 알기 어려움을 해결하기 위해, 위치 데이터를 중심으로 콘텐츠를 탐색할 수 있는 플랫폼을 설계하였습니다.

  • 게시물에 위치 정보와 좌표를 결합 → 지도 기반 탐색
  • Kakao Map SDK로 실제 장소 중심의 UI 제공
  • Oracle 기반 데이터 구조로 다중 사용자 요청 처리 안정화

주요 기능

  • 회원 관리: 회원가입, 로그인, 아이디/비밀번호 찾기, 마이페이지, 회원 탈퇴
  • 게시물 기능: 이미지 업로드, 위치 정보 입력, 최신순/인기순/팔로우 게시물 조회, 게시물 검색
  • 댓글 시스템: 게시물별 댓글 작성 및 조회
  • 지도 기능: Kakao Maps SDK 통합, 지도 클릭으로 마커 위치 설정, 게시물 위치 연동
  • 사용자 검색: 닉네임 기반 사용자 검색
  • 팔로우 시스템: 팔로우 및 팔로워 관리

기술 스택

  • Backend: Java (JSP/Servlet), Oracle Database 11g
  • Frontend: HTML5, CSS3, JavaScript, Bootstrap
  • Map API: Kakao Maps SDK
  • File Upload: COS (Multipart)
  • Server: Apache Tomcat 8.5
  • 아키텍처: MVC 패턴 (DTO/DAO), 커넥션 풀, 세션 관리

시스템 아키텍처

사용자 요청 → Front-End JSP/JS → Controller (Servlet) → Service Layer (DAO/DTO) → DB Connection Pool (Oracle)

기술적 도전 및 개선 경험

  • 파일 업로드 안정화: Multipart 처리 시 인코딩 깨짐 문제 → UTF-8 필터 설정으로 해결
  • SQL Injection 방지: PreparedStatement 적용, 검색 쿼리 문자열 연결 제거
  • API 응답 속도 개선: 인기 게시물 조회 SQL 인덱스 튜닝 (LIKE → INSTR + INDEX)

프로젝트 경험

이 프로젝트를 통해 첫 프로젝트 개발을 경험했습니다. 1인으로 프로젝트 개발의 흐름을 알게되었고, JSP/Servlet 기반의 MVC 패턴을 구현하며 웹 애플리케이션의 기본 구조와 동작 원리를 학습했습니다. 또한 Kakao Maps API를 활용해 보면서 외부 API 통합 경험을 할 수 있는 좋은 기회가 되었습니다.
